3. Como rodar no Colab

- Envie o notebook e os 6 arquivos dados-sinteticos-*.json para o Colab (ou suba os JSONs para /content).

- Execute as células na ordem: instalação → imports → carregar dados → anonimização.
(Opcional) Rodar a célula de fine-tuning (usa GPU e demora).

- Executar a célula de teste rápido ou chamar rodar_assistente_interativo() para usar o assistente com RAG + LangGraph.

- Se quiser, posso ajustar o notebook (por exemplo: outro modelo base, mais intenções no grafo ou mais dados sintéticos) ou revisar algum trecho específico.
